## How to run the app

- Clone the app
  ```
  git clone https://github.com/Shruthi-Sivagnanam/writings-blog.git
  ```

- Download the dependencies
  - For server side
     ```
    npm install 
    ```
  - For client side
    ```
    cd client
    npm install
    ```
 - Run the app
    ```
    npm run dev
    ```
